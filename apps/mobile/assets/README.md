# Assets

Drop the following PNGs here before publishing a build. Expo will fall back to
its default placeholder for missing files during local dev:

- `icon.png` — 1024×1024, used for iOS app icon and Expo Go listing.
- `splash.png` — 1242×2436 portrait, shown while the JS bundle loads.
- `adaptive-icon.png` — 1024×1024 foreground for Android adaptive icon.
- `notification-icon.png` — 96×96 monochrome PNG, white-on-transparent.

The brand colour `#6C63A6` is configured as the splash + adaptive icon
background in `app.json`.
